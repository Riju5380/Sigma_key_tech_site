# SIGMA KEY TECH Website

A modern, responsive website for **SIGMA KEY TECH**, a student safety and monitoring platform that helps parents stay connected with their child's school through real-time RFID attendance tracking, notifications, and academic updates.

The website serves as the official landing page for the application and provides users with information about the platform, privacy policy, support, terms of service, and account deletion procedures.

---

## 📌 Overview

SIGMA KEY TECH is designed to improve communication between schools and parents by providing instant access to important student activities.

The website showcases the platform's features and provides all required public pages for users, making it suitable for deployment as a product information website.
